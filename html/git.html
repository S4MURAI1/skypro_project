Git

  • git ls-files - это служебная команда, которая выводит список файлов, находящихся в индексе (stage) Git и рабочей директории. Она полезна для поиска отслеживаемых файлов и интеграции со скриптами.
  • cd - расшифровывается как "Change directory", что переводится как "сменить папку" или "перейти в другую папку".
  • git rev-parse --show-prefix - выводит путь от корня репозитория до вашей текущей папки.
  • mkdir - расшифровывается как "Make directory", что переводится как "создать папку".
  • git mv - расшифровывается как "Move", что переводится как "переместить", "передвинуть". С помощью нее можно переместить, переименовать файлы и папки
